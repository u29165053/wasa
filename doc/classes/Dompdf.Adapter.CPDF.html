CPDF

Implements \Dompdf\Canvas

PDF rendering interface

Dompdf\Adapter\CPDF provides a simple stateless interface to the stateful one provided by the Cpdf class.

Unless otherwise mentioned, all dimensions are in points (1/72 in). The coordinate origin is in the top left corner, and y values increase downwards.

See http://www.ros.co.nz/pdf/ for more complete documentation on the underlying \Dompdf\Adapter\Cpdf class.

Adds a specified 'object' to the document

add_object(integer $object, string $where = 'all')

$object int specifying an object created with {@link CPDF::open_object()}. $where can be one of:

  • 'add' add to current page only
  • 'all' add to every page from the current one onwards
  • 'odd' add to all odd numbered pages from now on
  • 'even' add to all even numbered pages from now on
  • 'next' add the object to the next page only
  • 'nextodd' add to all odd numbered pages from the next one
  • 'nexteven' add to all even numbered pages from the next one
see \Dompdf\Adapter\Cpdf::addObject()

Arguments

$where

While an object is open, all drawing actions are recored in the object, as opposed to being drawn on the current page. Objects can be added later to a specific page or to several pages.

The return value is an integer ID for the new object.
