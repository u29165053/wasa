HashManager

Clase que contiene métodos para calcular diferentes hashes de un fichero.

Esta clase se ayuda de un script desarrollado en Python 2.x, por lo que es necesario indicar la ruta de este script.

La ruta de dicho archivo python, se debe indicar en el constructor de la clase, en la variable "pythonScript"
