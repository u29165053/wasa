Factory¶

Contains frame decorating logic

This class is responsible for assigning the correct AbstractFrameDecorator, \Dompdf\Frame\AbstractPositioner, and \Dompdf\Frame\AbstractFrameReflower objects to Frame objects. This is determined primarily by the Frame's display type, but also by the Frame's node's type (e.g. DomElement vs. #text)

FrameTree¶

Represents an entire document as a tree of frames

The FrameTree consists of Frame objects each tied to specific DOMNode objects in a specific DomDocument. The FrameTree has the same structure as the DomDocument, but adds additional capabalities for styling and layout.
